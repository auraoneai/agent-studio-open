# Agent Studio Open VS Code Extension

This extension surface exposes:

- Workspace MCP server auto-discovery commands.
- Agent Studio activity-bar view with a manifest inspector tree.
- Compose webview for quick tool payload editing.
- Hover provider for risk findings on common destructive or financial MCP tools.
- Deep link command to continue record/replay work in the desktop app.

Marketplace target ID: `auraone.agent-studio`.
